# Netflix Landing/Profile Selector page Clone

A clone of the **Netflix Landing page and Profile Selector** made with **React** and **Tailwind CSS**.

Allows the user to choose between different mocked profiles.

## Features

- Profile selection with smooth animations.

- Responsive and modern design.

- Profile states managed with React Hooks (`useState`).

- Easy to expand to add new profiles.

## Technologies

- Reactjs

- Tailwind CSS

- Modern JavaScript (ES6+)

## How to run locally

1. Clone the repository:
```bash
`git clone https://github.com/LeonardoFCorrea/netflix-landingPage-profileSelector.git`
```
2. Enter the project main folder:
```bash
`cd netflix-landingPage-profileSelector`
```
3.Install dependencies:
```bash
`npm install`
```
4. Run project:
```bash
`npm start`
```
